Alur Kerja
Project Workflow
1
Dataset Generation & Identifikasi Masalah
Dataset sintetis 2.500 baris dibuat dengan sengaja memiliki dirty data: duplikat, format tanggal tidak konsisten, nilai negatif, karakter non-numerik, dan inkonsistensi kapitalisasi.
🐍 Python · random · csv
2
Data Cleaning dengan Python & Pandas
Script data_cleaning.py membersihkan 10 duplikat, 247 tanggal invalid, 193 harga kotor, 244 qty tidak valid. Output: 1.806 baris bersih + kolom turunan.
🧹 Pandas · NumPy · Regex
3
Analisis dengan SQL Server (T-SQL)
Data bersih diimpor ke SQL Server. 10 query analitik dibuat mencakup KPI ringkasan, tren MoM, ranking produk & salesperson, analisis YoY, dan kontribusi per wilayah.
🗄️ T-SQL · Window Functions · CTE · BULK INSERT
4
Visualisasi di Looker Studio
Dashboard interaktif di Google Looker Studio menampilkan 4 KPI tile, tren bulanan, distribusi kategori, top produk, heatmap wilayah, dan leaderboard salesperson.
📊 Looker Studio · Charts · Filters · Slicers

Cuplikan Data Cleaning (Python)
# ── Parsing multi-format tanggal ──────────────────────────── def parse_date(val): if pd.isna(val) or str(val).strip() in ("", "N/A"): return pd.NaT formats = ["%Y-%m-%d", "%d/%m/%Y", "%m-%d-%Y", "%Y%m%d"] for fmt in formats: try: return pd.to_datetime(val, format=fmt) except: continue return pd.NaT # ── Bersihkan unit price (hapus "Rp", koma, simbol) ───────── def clean_price(val): val = re.sub(r"[Rp\s,]", "", str(val)) try: v = float(val) return v if v > 0 else np.nan except: return np.nan # ── Imputasi rating dengan median per produk ───────────────── df["rating"] = df.groupby("product_name")["rating"].transform( lambda x: x.fillna(x.median()) ) # ── Kolom turunan: net_sales ────────────────────────────────── df["net_sales"] = (df["unit_price"] * df["quantity"] * (1 - df["discount_percent"] / 100))
Kode
Cuplikan SQL — MoM Growth & Ranking
-- Tren bulanan + Month-over-Month Growth SELECT order_year, order_month, order_month_name, SUM(net_sales) AS net_sales, -- MoM Growth menggunakan Window Function LAG SUM(net_sales) - LAG(SUM(net_sales)) OVER (ORDER BY order_year, order_month) AS mom_growth, CAST(100.0 * (SUM(net_sales) - LAG(SUM(net_sales)) OVER (ORDER BY order_year, order_month)) / NULLIF(LAG(SUM(net_sales)) OVER (ORDER BY order_year, order_month), 0) AS DECIMAL(6,2)) AS mom_pct FROM dbo.sales_transactions GROUP BY order_year, order_month, order_month_name ORDER BY order_year, order_month;
Insight
Key Findings
💡
Laptop Pro X1 adalah produk unggulan
Menyumbang Rp 260,1 Miliar atau sekitar 31% dari total net sales, jauh melampaui produk lainnya. Strategi upsell produk ini sangat potensial untuk meningkatkan revenue.
📍
Jawa Barat + Jawa Timur = 42% transaksi
Dua wilayah ini mendominasi volume order. Potensi ekspansi masih terbuka lebar di Kalimantan Timur dan Sulawesi Selatan yang baru berkontribusi ~9.5% masing-masing.
🔄
Online & Offline hampir seimbang (52% vs 48%)
Distribusi channel yang seimbang mengindikasikan strategi omnichannel yang baik. Investasi lebih pada channel online dapat mendorong pertumbuhan lebih efisien.
⚠️
Data mentah: 27.8% baris perlu pembersihan
Dari 2.500 baris, 694 baris dieliminasi karena data kotor. Implementasi validasi data real-time di sistem point-of-sale dapat mencegah masalah kualitas data di masa depan.
